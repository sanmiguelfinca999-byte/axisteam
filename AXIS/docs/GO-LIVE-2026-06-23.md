## Si algo falla — diagnóstico rápido

| Síntoma | Probable causa | Fix |
|---|---|---|
| Login dice "Credenciales inválidas" | Cuenta no creada o password copiada con espacio | Verificar en Supabase Auth → Users; re-correr `setup-accounts.mjs` (es idempotente si fallas tienes que borrar manualmente) |
| Pantalla queda en "Conectando al Execution OS..." | Env vars de Vercel no aplicadas | Verificar redeploy completó; abrir DevTools → Network → ver request a Supabase |
| Operator ve datos de otro Operator | RLS no aplicada | Verificar en Supabase → Authentication → Policies que hay 14+ policies activas |
| `setup-accounts.mjs` da error 401 | service_role key mal copiada | Re-copiar de API settings (NO la anon) |
| Página blanca en `axisteam.vercel.app` | Build falló | Vercel → Deployments → ver log del último build |
